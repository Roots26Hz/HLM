# data/

Most weeks use scikit-learn's bundled toy datasets (`load_iris`, `load_wine`) and need
**no files here**. This folder is reserved for later weeks that introduce CSV datasets.

When you add a CSV:

1. Drop the file here (e.g. `data/housing.csv`).
2. Document its **provenance** below — where it came from and how to regenerate/fetch it,
   so the repo stays reproducible for anyone who clones it.
3. Load it in a notebook with a path relative to the repo root, e.g.
   `pd.read_csv("../../data/housing.csv")`, or add a loader to `shared/ml_utils`.
